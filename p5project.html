P5 Project (2020)

P5 is een open-source library voor Javascript.
Ik maak samen met een klasgenoot een spel dat zichzelf kan spelen.

Dit spel is gebaseerd op het Google Chrome Dinosaurus-spel, waar een dinosaurus over een oneindig aantal cactussen springt.

We hebben een GitHub repository gemaakt, "dino-sauce" en de eerste bestanden zijn opgesteld. De concept art voor het spel is al gemaakt!

Een paar weken later hebben wij veel toegevoegd aan het project. Onder andere een werkend menu, score, en het meest opmerkelijke een experimentele AI-mode.
Mijn klasgenoot doet voor het grootste deel het programmeerwerk, terwijl ik voor het grafisch ontwerp en geluid zorg.
